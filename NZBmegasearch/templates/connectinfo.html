Connection to SICKBEARD 

Sickbeard connectivity is always active.

  • In Sickbeard: go to Config/Search Providers and add a custom Newznab provider...
    On build 501 (2013-10-06) or newer enter 0 (zero) else no API key is needed (unless explicitly set in the config)

Connection to COUCHPOTATO 

Couchpotato connectivity is always active.

  • In Couchpotato: go to Settings/Providers and add a custom Newznab provider...
    Use API key 123 (any key number will work unless explicitly set in the config)

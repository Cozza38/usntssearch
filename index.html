NZBmegasearcH

NZBmegasearcH is a program that aggregates all your (not only) newznab accounts to one place.
Are you tired to open multiple browser windows for searching with different sites and cross-checking the results? Well, that's the solution for you. NZBmegasearcH automatically collects search results from all your accounts in one clean web interface.

Technically speaking: NZBmegasearcH is a meta-search engine that retrieves data from your collection of NZB websites.

... and that's more: you can configure to use it with your SB and CP as unique NZB search provider.

Features:

  • Support for accounts from any Newsnb and Spotweb based provider
  • Support for web-only access to Newsnb websites (in case API is not active)
  • Support for nzb.cc, wombie, omgwtfnzbs, gingadaddy and many others
  • Support for CP, SB, NZB360, SABNZBD, NZBGET
  • Detects PREDB releases
  • Support for many search categories
  • Retrieves trending movies and shows
  • Gives appropriate search suggestions
  • Small memory and CPU footprint
  • SSL and user identification supported

It is written in Python and it works on linux, mac, win. Windows binaries are also available.

Latest version is 0.44

This project makes use of:

  • Python 2.7
  • Flask
  • Requests
  • BeautifulSoup
  • Mechanize
  • PyOpenSSL
